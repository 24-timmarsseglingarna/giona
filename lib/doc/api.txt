API specification - version 1

SIGN IN
curl -H "Content-Type: application/json" -X POST -d '{ "user": { "email": "someone@somewhere.com", "password": "secret" } }' http://localhost:8080/users/sign_in

REQUEST HEADERS
X-User-Email
X-User-Token

RESPONSE HEADERS
X-User-Email
X-User-Id
X-User-Token

QUERIES
/api/v1/class?parameter1=value1&parameter2=value2

CLASSES
regattas?is_active=boolean&has_race=N
races?from_regatta=N&has_team=O
teams?from_race=N&from_boat=O&has_person=P
people?from_team=N&has_user=O
user.from_person=N
boats?has_class=N&from_team=O
boat_classes?from_boat
