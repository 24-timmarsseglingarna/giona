API specification - version 1

SIGN IN
curl -H "Content-Type: application/json" -X POST -d '{ "user": { "email": "someone@somewhere.com", "password": "secret" } }' http://localhost:8080/users/sign_in.json

REQUEST HEADERS
X-User-Email
X-User-Token

RESPONSE HEADERS
X-User-Email
X-User-Id
X-User-Token
